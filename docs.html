S - Single Responsibility

Each service handles one specific responsibility. WebsiteCreationService only creates websites, PostRetrievalService only retrieves posts.

O - Open/Closed

Services can be extended without modification. Add caching by implementing CachedPostRetrievalService without changing existing code.

L - Liskov Substitution

Interface implementations are substitutable. Any PostRetrievalInterface implementation can replace PostRetrievalService.

I - Interface Segregation

Focused, cohesive interfaces. Separate interfaces for creation and retrieval operations instead of one fat interface.

D - Dependency Inversion

Dependencies on abstractions, not concretions. Controllers depend on interfaces, not concrete service classes.
